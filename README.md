# Anun Dictionary Resources

A comprehensive database for the **Anun conlang**, structured as an English-Anun learner's guide. This repository serves as both a development space and a source of truth for Anun language resources.

## 📚 Structure

```
Anun-Dictionary-Resources/
├── README.md (this file)
├── data/
│   ├── vocabulary/
│   │   ├── nouns.json
│   │   ├── verbs.json
│   │   ├── adjectives.json
│   │   ├── adverbs.json
│   │   ├── prepositions.json
│   │   └── particles.json
│   ├── grammar/
│   │   ├── phonology.json
│   │   ├── inflections.json
│   │   ├── syntax.json
│   │   └── cases.json
│   ├── texts/
│   │   ├── sample_texts.json
│   │   └── practice_sentences.json
│   └── index.json (master vocabulary index)
├── docs/
│   ├── CONVENTIONS.md (formatting and naming conventions)
│   └── STRUCTURE.md (detailed explanation of data structure)
└── tools/ (future: scripts for processing data)
```

## 🚀 Getting Started

This project is organized to be:
- **Human-readable**: Easy to browse and edit in GitHub
- **Machine-readable**: JSON format for programmatic access
- **Learner-friendly**: Progressive vocabulary organized by frequency and difficulty
- **Scalable**: Grows naturally from vocabulary → grammar → texts

## 💡 Contributing

- Part of speech
- Example sentences
